Sugerencia tecnica para el proyecto
- Mantener ids cortos de filtro (ej: none, vivid, mono, warm, cool, noir, clarendon).
- Guardar en config central por filtro:
  - id
  - label
  - cssFilter (preview rapido)
  - shader/pipeline (futuro)
  - parametros editables (intensidad 0-100)
- Permitir intensidad por slider para no multiplicar botones.
- En una fase avanzada, migrar de CSS filter a WebGL/WebGPU para filtros complejos.
